CÓDIGO

PRODUCTO

C2

Que se tome conocimiento por los medios de difusión pública u otro, según sea el caso, que un cliente está siendo investigado o procesado por el delito de lavado de activos, delitos precedentes, el delito de financiamiento del terrorismo y/o delitos conexos.

Todos los productos


Monitoreo Señales de alerta C2
  1. Se realiza la búsqueda del Registro Negativo y base de PEP y Familiares de PEP contra la base de clientes con la finalidad de identificar la señal de alerta en cuestión. Para esto, la Oficialía de Cumplimiento es responsable de enviar la lista de registro.
  2. Complementariamente, esta señal de alerta se revisa mediante el “Procedimiento para el monitoreo de Listas Especiales de Seguridad” según lo indicado en el documento Sistema de Monitoreo de Operaciones – SPLAFT.
  3. Complementariamente, esta señal de alerta se revisa mediante la búsqueda de clientes con información negativa en fuentes públicas.
  4. En caso se identifique señal de alerta y no sea descartado como falso positivo, se procederá a analizar la inusual operación.
